## 📖 Overview

**LuaJava Layout Creator Pro** is a browser-based tool designed for Game Guardian script developers who want to create professional-looking LuaJava floating menus without writing layout code manually. 

With an intuitive visual interface, you can customize every aspect of your script's UI - from colors and dimensions to switch behaviors and padding - and see changes reflected in real-time. The tool generates complete, production-ready `.lua` scripts compatible with Game Guardian.

## ✨ Features

### 🎨 **Visual Customization**
- **Live Preview**: See your layout changes instantly in a simulated Android environment
- **Color Management**: Full control over backgrounds, text, switches, buttons, and accents with color pickers and hex inputs
- **Typography**: Adjust title, subtitle, switch labels, and button text sizes independently
- **Interactive Preview**: Click switches in the preview to toggle their states

### 📐 **Layout Control**
- **Responsive Dimensions**: Set width/height as percentage of screen with sliders
- **Padding & Margins**: Fine-tune spacing for main container, cards, switches, and buttons
- **Border Settings**: Configure border radius, width, and corner rounding for every element
- **Switch Customization**: Adjust thumb size, track radius, min width, and individual padding per switch

### 🔧 **Function Management**
- **Add/Edit/Delete Functions**: Create hack functions with ON/OFF code blocks
- **Duplicate Functions**: Quickly clone existing functions
- **Function Naming**: Separate internal function names from display labels
- **Code Templates**: Pre-loaded with common Game Guardian search patterns

### 💾 **Export & Import**
- **Download .lua Scripts**: Generate complete, ready-to-use LuaJava scripts
- **Copy to Clipboard**: Quick copy for immediate pasting
- **Export Settings**: Save your configuration as JSON for backup
- **Import Settings**: Restore previous configurations instantly
- **Keyboard Shortcuts**: `Ctrl+S` to download, `Ctrl+C` to copy code

### 🌐 **Browser Features**
- **No Installation Required**: Works entirely in the browser
- **Responsive Design**: Adapts to desktop, tablet, and mobile screens
- **Persistent Layout**: Organized tabs keep settings accessible without scrolling overload
- **Toast Notifications**: Visual feedback for all actions
